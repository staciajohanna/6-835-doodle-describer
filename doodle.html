How to draw:
2 hands on Leap
- Left hand:
open palm = start stroke, close fist = end stroke
- Right hand: pen stroke
1 hand on Leap + spacebar
- Spacebar: pressed = start stroke, not pressed = end stroke
- Right hand: pen stroke
Mouse
Detach the leap from your computer, then use the mouse for drawing.
Voice Command:
- "submit" = Submit drawing to be analyzed
- "clear" = Clear canvas
